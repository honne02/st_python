## Что покажет приведенная ниже программа?

```python
def change_us(a, b):
    a = 0
    b = 0
    print(a, b)

x = 1
y = 7
print(x, y)
change_us(x, y)
print(x, y)
```

```
1 7
0 0
1 7
```

## Что покажет приведенная ниже программа?

```python
def print_text(text, num):
    while num > 0:
        print(text, end='')
        num -= 1

print_text('Python', 4)
```

```
PythonPythonPythonPython
```
